# Expense Tracker

A full-stack expense tracker built with React, Vite, Tailwind CSS, Express, MongoDB, and Recharts.

## Features

- Dashboard with income, expense, savings, and recent transaction overview
- Add, edit, delete, filter, and export income records
- Add, edit, delete, filter, and export expense records
- Time-frame filters for daily, weekly, monthly, and yearly views
- Excel export for income and expense data
- Works with MongoDB, with a temporary in-memory fallback if MongoDB is not running

## Run Locally

Open two terminals from the project root.

Backend:

```bash
cd backend
npm start
```

Frontend:

```bash
cd frontend
npm run dev
```
